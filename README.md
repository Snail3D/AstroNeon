# AstroNeon

A modern, neon-styled Asteroids game built with HTML5 Canvas.

![AstroNeon](https://img.shields.io/badge/Platform-Web-brightgreen)
![License](https://img.shields.io/badge/License-MIT-blue)

## 🎮 Play Now

Open [index.html](https://snail3d.github.io/AstroNeon/) directly in your browser, or clone and open locally.

## Controls

| Key | Action |
|-----|--------|
| **W** | Thrust forward |
| **S** | Reverse thrust |
| **A** | Strafe left |
| **D** | Strafe right |
| **SPACE** | Fire |
| **Mouse** | Aim |

## Features

- 🚀 Smooth physics with acceleration and friction
- 💥 Particle explosions when asteroids are destroyed
- ✨ Neon glow effects (cyan, magenta, yellow)
- 🔄 Screen wrapping for ship and asteroids
- 🛡️ Invulnerability frames after respawning
- 📈 Score tracking with increasing difficulty
- 🌙 Beautiful starfield background
